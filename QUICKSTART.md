# BioTransLM 可视化系统 - 5 分钟快速开始指南

## 🚀 第一步：安装依赖（2 分钟）

```bash
# 进入项目目录
cd c:\Users\16779\Desktop\BioTransLM

# 安装 UI 额外依赖（如果已有 torch 等基础库）
pip install -r requirements-ui.txt

# 或者手动安装核心依赖
pip install gradio plotly networkx pandas
```

**预期输出**：
```
Successfully installed gradio-4.x.x plotly-5.x.x networkx-3.x pandas-2.x
```

---

## 🎯 第二步：启动应用（1 分钟）

```bash
# 方式一：使用启动脚本（推荐）
python run_ui.py

# 方式二：直接运行
python ui/app.py
```

**预期输出**：
```

============================================================

访问地址：http://127.0.0.1:7860
```

浏览器会**自动打开** http://127.0.0.1:7860

## 💡 第三步：体验核心功能（2 分钟）

### 1️⃣ 试试对话功能

1. 点击 **"💬 对话"** 标签页
2. 在输入框输入：`你好，请介绍一下你自己`
3. 点击 **"📤 发送"** 或按回车
4. 等待模型生成响应

**提示**：可以调节 Temperature 参数让回答更有创造性！

---

### 2️⃣ 查看处理过程

1. 点击 **"🔍 处理过程查看"** 标签页
2. 在下拉菜单中选择刚才的对话
3. 展开各个阶段查看详细信息：
   - HTM 系统激活热图
   - 记忆检索结果
   - 双系统推理过程

---

### 3️⃣ 加载示例数据集

1. 点击 **"📊 数据集管理"** 标签页
2. 在右侧选择模板类型：**"一问一答"**
3. 点击 **"加载模板数据"**
4. 查看预览和数据统计

---

### 4️⃣ 模拟训练监控

1. 点击 **"📈 训练监控"** 标签页
2. 展开 **"训练配置"** 设置参数
3. 点击 **"▶️ 开始训练"**
4. 观察损失曲线和 HTM 指标变化

**注意**：当前是演示模式，真实训练需要先加载数据集。

## ⚙️ 常见问题速查

### 问题 1：端口被占用

**错误信息**：`Address already in use for port: 7860`

**解决方案**：
```bash
# 使用其他端口
python run_ui.py --port 8080
```

---

### 问题 2：依赖安装失败

**错误信息**：`No module named 'gradio'`

**解决方案**：
```bash
# 升级 pip
python -m pip install --upgrade pip

# 重新安装
pip install gradio plotly networkx pandas
```

---

### 问题 3：浏览器没有自动打开

**原因**：某些系统配置阻止了自动打开

**解决方案**：
- 手动访问：http://127.0.0.1:7860
- 或者添加 `--no-browser` 参数

---

### 问题 4：对话无响应

**可能原因**：
- Orchestrator 初始化失败
- 缺少依赖文件

**解决方案**：
1. 检查控制台错误日志
2. 确保所有文件完整
3. 运行测试脚本：`python test_ui_components.py`

## ✨ 高级技巧

### 技巧 1：导出对话记录

在对话标签页点击 **"💾 导出对话"**，保存为 JSON 格式，方便后续分析。

### 技巧 2：自定义数据集

1. 先在 "数据集管理" 加载数据
2. 然后到 "训练监控" 启动训练
