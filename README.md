## ✨ 核心特性

### 🧠 生物启发式架构

```
人脑特征              BioTransLM 实现
─────────────────────────────────────
① 稀疏激活     →     SDR (2% 激活率)
② 分层处理     →     7 步处理流程
③ 时序记忆     →     HTM 时序记忆器
④ 双系统思考   →     系统 1(直觉) + 系统 2(推理)
```

### 🔍 完全可解释

传统 AI：输入 → [黑盒] → 输出  
BioTransLM：输入 → ①感觉门控 → ②编码 → ③HTM → ④调节 → ⑤记忆 → ⑥推理 → ⑦生成 → 输出

### 🎨 现代化界面

- 💬 简洁风格对话界面
- 📊 实时训练监控
- 🔍 处理过程可视化
- 📈 智能推荐配置

## 🚀 快速开始

### 1️⃣ 安装依赖

```bash
pip install -r requirements.txt
pip install -r requirements-ui.txt
```

### 2️⃣ 启动界面

**Windows 用户**:
双击 `启动界面.bat`

**其他系统**:
```bash

```

### 3️⃣ 打开浏览器

访问：http://127.0.0.1:7860

## 🏗️ 技术架构

### 核心技术栈

- **PyTorch** - 深度学习框架
- **Gradio** - Web UI 框架
- **Plotly** - 数据可视化
- **Pandas** - 数据处理

### 理论基础

- **HTM 理论** - Hierarchical Temporal Memory
- **SDR** - Sparse Distributed Representation
- **双系统理论** - System 1 (直觉) + System 2 (推理)

### 项目结构

```
BioTransLM/
├── core/           # 核心模块
├── generation/     # 生成模块
├── htm/           # HTM 系统
├── memory/        # 记忆系统
├── reasoning/     # 推理系统
├── ui/            # 可视化界面 ⭐
│   ├── tabs/      # 标签页
│   ├── components/# 组件
│   └── utils/     # 工具类
├── 使用说明书.md   📘
├── LICENSE.md     📜
└── 启动界面.bat    🚀
```

## 🎓 学习路线

### 第 1 天：熟悉界面
1. 启动程序
2. 浏览 5 个标签页
3. 阅读《使用说明书》

1. 加载预训练模型（如果有）
2. 在"对话"页面测试
3. 在"处理过程查看"观察

1. 提出改进想法
2. 修改代码验证
