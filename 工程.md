# BioTransLM 生物启发式AI架构工程文档

## 项目概述

BioTransLM是一个**完全非Transformer**的AI架构，模拟人类大脑认知过程。采用稀疏编码、分层时序记忆(HTM)、脉冲神经网络(SNN)和双系统推理，实现类人的文本理解与生成。

**核心特性**：
- ✅ 无Transformer依赖，纯生物启发机制
- ✅ 兼容Transformer训练数据集格式
- ✅ 单一训练范式，多层级生物启发式学习
- ✅ 稀疏计算，能耗降低~90%

## 模块详细设计（纯生物启发版）

### 1. 输入处理层 - 无Transformer依赖

#### 1.1 嵌入转换层 (EmbeddingConverter)

**功能描述**：
- 将标准Transformer词嵌入（如BERT/GPT）转换为HTM/SNN兼容格式
- 或直接训练原生稀疏嵌入
- 保持词汇表兼容性，支持直接使用token ids输入

**核心实现**：
```python
class EmbeddingConverter(torch.nn.Module):
    """
    将标准词嵌入转换为HTM兼容的稀疏表示
    
    无Transformer，仅用线性层+稀疏化激活
    """
    def __init__(self, vocab_size=30522, input_dim=768, 
                 output_dim=2048, sparsity_target=0.95):
        super().__init__()
        self.vocab_size = vocab_size
        self.sparsity_target = sparsity_target
        
        # 原生嵌入层（可选：也可加载预训练嵌入）
        self.native_embedding = torch.nn.Embedding(vocab_size, input_dim)
        
        # 转换层：将嵌入转换为高维稀疏表示
        self.converter = torch.nn.Sequential(
            torch.nn.Linear(input_dim, output_dim),
            torch.nn.ReLU(),
            SparsifyActivation(sparsity_target)  # 自定义稀疏激活
        )
    
    def forward(self, input_ids=None, input_embeddings=None):
        """
        支持两种输入模式：
        1. input_ids: token id序列（兼容Transformer数据格式）
        2. input_embeddings: 预训练词嵌入（如BERT输出）
        """
        if input_embeddings is None:
            input_embeddings = self.native_embedding(input_ids)
        
        sparse_output = self.converter(input_embeddings)
        return sparse_output

class SparsifyActivation(torch.nn.Module):
    """稀疏激活层：只保留top-k激活值"""
    def __init__(self, sparsity_target=0.95):
        super().__init__()
        self.k_ratio = 1 - sparsity_target  # 保留的激活比例
    
    def forward(self, x):
        k = int(x.size(-1) * self.k_ratio)
        # 找到top-k激活
        topk_values, topk_indices = torch.topk(x, k, dim=-1)
        # 创建稀疏掩码
        output = torch.zeros_like(x)
        output.scatter_(-1, topk_indices, topk_values)
        return output
```

**关键指标**：
- 输入格式兼容性: 100%（与HuggingFace Tokenizer一致）
- 输出稀疏度: 95% (仅5%神经元激活)
- 转换延迟: <5ms/batch

#### 1.2 感觉门控模块 (SensoryGating)

**功能描述**：
- 模拟大脑丘脑网状核的门控机制
- 过滤输入噪声，识别关键信息
- 动态调整注意力阈值
- 基于任务相关性进行信息筛选

**核心算法**：
```python
class SensoryGating:
    def __init__(self, config):
        self.config = config
        self.base_threshold = config.threshold
        self.noise_reduction = config.noise_reduction
        self.attention_history = []
        
    def forward(self, sparse_activations, attention_mask=None, task_context=None):
        """
        输入: 稀疏激活张量 [batch, seq_len, n_columns]
        输出: 门控后的稀疏激活
        处理: 噪声估计 -> 阈值计算 -> 门控
        """
        batch_size, seq_len, _ = sparse_activations.shape
        
        # 1. 噪声估计（基于低激活区域）
        noise_level = self._estimate_noise(sparse_activations)
        
        # 2. 动态阈值计算
        if task_context is not None:
            # 根据任务类型调整阈值
            task_threshold = self._task_dependent_threshold(task_context)
        else:
            task_threshold = self.base_threshold
            
        final_threshold = task_threshold * (1 + noise_level * self.noise_reduction)
        
        # 3. 应用门控
        gated = sparse_activations.clone()
        gated[gated < final_threshold] = 0
        
        # 4. 应用注意力掩码（如提供）
        if attention_mask is not None:
            gated = gated * attention_mask.unsqueeze(-1)
        
        # 5. 保持稀疏度
        gated = self._maintain_sparsity(gated)
        
        return {
            'gated_activations': gated,
            'noise_level': noise_level,
            'threshold_used': final_threshold,
            'attention_weights': self._compute_attention(gated)
        }
    
    def _estimate_noise(self, activations):
        """估计噪声水平：低激活值的方差"""
        low_activation_mask = activations < torch.quantile(activations, 0.7)
        if low_activation_mask.any():
            noise_est = activations[low_activation_mask].std()
        else:
            noise_est = 0.0
        return torch.sigmoid(noise_est)
```

**关键指标**：
- 噪声过滤率: >85%
- 关键信息保留率: >95%
- 处理延迟: <10ms

#### 1.3 多粒度编码模块 (MultiGranularEncoder)

**技术栈（无Transformer）**：
- 字符级：CNN卷积网络提取字形特征
- 词级：双向LSTM/RNN提取上下文特征
- 短语级：n-gram卷积+池化
- 句子级：层次化池化+全局特征提取

**核心实现**：
```python
class MultiGranularEncoder(torch.nn.Module):
    """无Transformer的多粒度编码器，使用CNN+RNN组合"""
    
    def __init__(self, config):
        super().__init__()
        self.config = config
        self.hidden_dim = config.hidden_dim  # e.g., 256
        self.output_dim = config.output_dim  # e.g., 1024 (HTM列数)
        
        # 1. 字符级CNN
        self.char_cnn = CharCNN(
            vocab_size=config.vocab_size,
            embed_dim=64,
            filters=[(2, 64), (3, 64), (4, 64)]  # n-gram滤波器
        )
        
        # 2. 词级BiLSTM
        self.word_rnn = torch.nn.LSTM(
            input_size=config.embed_dim,
            hidden_size=self.hidden_dim,
            num_layers=2,
            bidirectional=True,
            batch_first=True,
            dropout=0.1
        )
        
        # 3. 短语级CNN
        self.phrase_cnn = PhraseCNN(
            input_dim=self.hidden_dim * 2,
            filters=[(2, 128), (3, 128), (5, 64)]
        )
        
        # 4. 融合层
        fusion_dim = 64*3 + self.hidden_dim*2 + 128+128+64
        self.fusion = torch.nn.Sequential(
            torch.nn.Linear(fusion_dim, self.output_dim),
            torch.nn.ReLU(),
            SparsifyActivation(sparsity_target=0.95)
        )
    
    def forward(self, input_ids, char_inputs=None):
        batch_size, seq_len = input_ids.shape
        
        # 基础嵌入
        embeds = self.embedding(input_ids)  # [B, T, D]
        
        # 多尺度特征提取
        char_features = self.char_cnn(char_inputs) if char_inputs else \
                       self.char_cnn(embeds)  # [B, T, 192]
        word_features, _ = self.word_rnn(embeds)  # [B, T, 512]
        phrase_features = self.phrase_cnn(word_features)  # [B, T, 320]
        
        # 特征融合
        combined = torch.cat([char_features, word_features, phrase_features], dim=-1)
        sparse_output = self.fusion(combined)
        
        return {
            'char_level': char_features,
            'word_level': word_features,
            'phrase_level': phrase_features,
            'combined': sparse_output,
            'hierarchy_graph': self._build_hierarchy(char_features, 
                                                   word_features, 
                                                   phrase_features)
        }
```

### 2. 分层时序记忆系统 (HTM) - 核心实现

#### 2.1 空间池化器 (SpatialPooler)

**功能描述**：将输入转换为稀疏分布式表示(SDR)

**核心参数配置**：
| 参数 | 值 | 说明 |
|------|----|------|
| 输入维度 | 1024 | 稀疏表示维度 |
| 微型柱数量 | 4096 | HTM基础计算单元 |
| 每柱细胞数 | 16 | 序列记忆容量 |
| 激活阈值 | 15 | 列激活所需突触数 |
| 学习阈值 | 10 | 触发学习所需突触数 |
| 潜在连接密度 | 0.05 | 初始连接稀疏度 |
| 激活率目标 | 0.02 | ~2%列激活 |

**算法实现**：
```python
class SpatialPooler:
    """HTM空间池化核心算法"""
    
    def __init__(self, config):
        self.config = config
        self.input_dim = config.input_dim
        self.n_columns = config.n_columns
        self.activation_threshold = config.activation_threshold
        self.learning_threshold = config.learning_threshold
        
        # 连接持久度矩阵 [n_columns, input_dim]
        self.permanences = torch.rand(n_columns, input_dim) * 0.5
        self.permanence_inc = config.permanence_inc  # +0.1
        self.permanence_dec = config.permanence_dec  # -0.05
        self.permanence_thresh = config.permanence_thresh  # 0.5
        
        # 连接权重：持久度>阈值则连接
        self.connections = (self.permanences > self.permanence_thresh).float()
        
    def compute_overlap(self, input_sdr):
        """计算每个微型柱的重叠分数"""
        # input_sdr: [batch, seq_len, input_dim]
        batch_size, seq_len, _ = input_sdr.shape
        
        # 计算输入激活位置
        input_active = (input_sdr.abs() > 1e-8).float()
        
        # 计算重叠度：[batch, seq_len, n_columns]
        overlap = torch.einsum('bti,ci->btc', input_active, self.connections)
        
        return overlap
    
    def inhibit_columns(self, overlap):
        """应用抑制，只保留top-k激活列"""
        batch_size, seq_len, n_cols = overlap.shape
        k = int(n_cols * self.config.activation_rate)  # 2%激活
        
        # 找出每个样本top-k激活列
        topk_values, topk_indices = torch.topk(overlap, k, dim=-1)
        
        # 创建激活掩码
        active_mask = torch.zeros_like(overlap)
        active_mask.scatter_(-1, topk_indices, 1.0)
        
        # 只保留超过阈值的激活
        threshold_mask = overlap > self.activation_threshold
        active_mask = active_mask * threshold_mask
        
        return active_mask, topk_values
    
    def learn(self, input_sdr, active_columns):
        """Hebbian学习：加强激活连接，减弱非激活连接"""
        input_active = (input_sdr.abs() > 1e-8).float()
        
        # 计算每个激活列的更新
        for col in range(self.n_columns):
            # 该列激活的样本
            col_active = active_columns[:, :, col].unsqueeze(-1)
            
            if col_active.sum() > 0:
                # 加强激活输入的连接：+inc
                inc = col_active * input_active * self.permanence_inc
                # 减弱未激活输入的连接：-dec
                dec = col_active * (1 - input_active) * (-self.permanence_dec)
                
                # 更新持久度
                self.permanences[col] += (inc + dec).sum(dim=[0, 1])
                
        # 截断到[0, 1]范围
        self.permanences.clamp_(0, 1)
        
        # 更新连接权重
        self.connections = (self.permanences > self.permanence_thresh).float()
    
    def forward(self, input_sdr, learn=True):
        """前向传播+可选学习"""
        overlap = self.compute_overlap(input_sdr)
        active_columns, overlap_values = self.inhibit_columns(overlap)
        
        if learn:
            self.learn(input_sdr, active_columns)
        
        # 输出稀疏激活
        output = active_columns * overlap_values
        
        return {
            'active_columns': active_columns,
            'overlap_scores': overlap_values,
            'sparse_output': output
        }
```

#### 2.2 时序记忆 (TemporalMemory)

**功能描述**：学习序列模式，预测未来激活

**核心数据结构**：
- 每个细胞有多个远端段（distal segments）
- 每个段连接到前一时刻的激活细胞
- 段激活预测细胞的激活

```python
class TemporalMemory:
    """HTM时序记忆核心"""
    
    def __init__(self, config):
        self.config = config
        self.n_columns = config.n_columns
        self.n_cells_per_col = config.n_cells_per_col
        self.total_cells = self.n_columns * self.n_cells_per_col
        
        # 段阈值：激活一个段需要多少突触
        self.segment_threshold = config.segment_threshold
        
        # 预测与状态历史
        self.prev_predicted = None
        self.prev_active_cells = None
        
        # 段连接：稀疏矩阵表示
        self.segments = []  # 动态增长的段列表
        self.segment_connections = {}  # segment_id -> 连接细胞
        
        # 细胞选择
        self.cell_usage = torch.zeros(self.n_columns, self.n_cells_per_col)
        
    def compute_prediction(self, active_cells_prev):
        """基于前一状态计算当前预测"""
        n_segments = len(self.segments)
        if n_segments == 0:
            # 初始化：随机预测
            return torch.zeros(active_cells_prev.shape[0], 
                               active_cells_prev.shape[1],
                               self.n_columns, self.n_cells_per_col)
        
        # 计算每个段的激活
        segment_activity = torch.zeros(active_cells_prev.shape[0], 
                                       active_cells_prev.shape[1],
                                       n_segments)
        
        for i, seg in enumerate(self.segments):
            # 计算段连接细胞的激活和
            conn = self.segment_connections[i]
            # 简化实现：实际应使用稀疏计算优化
            act = active_cells_prev[:, :, conn].sum(dim=-1)
            segment_activity[:, :, i] = act
        
        # 段激活超过阈值则预测对应细胞激活
        predictions = torch.zeros(active_cells_prev.shape[0], 
                                  active_cells_prev.shape[1],
                                  self.n_columns, self.n_cells_per_col)
        
        for i, seg in enumerate(self.segments):
            col, cell = seg['cell_idx']
            seg_active = segment_activity[:, :, i] > self.segment_threshold
            predictions[:, :, col, cell] = seg_active.float()
        
        return predictions
    
    def burst_column(self, col_idx):
        """列为预测则burst（所有细胞激活），并选择学习细胞"""
        # 选择最少使用的细胞来学习新模式
        least_used = self.cell_usage[col_idx].argmin()
        return least_used
    
    def forward(self, active_columns_prev, active_columns_curr, learn=True):
        """
        active_columns_prev: 前一时刻激活列 [batch, seq_len, n_columns]
        active_columns_curr: 当前时刻激活列 [batch, seq_len, n_columns]
        """
        batch_size, seq_len, _ = active_columns_curr.shape
        
        # 1. 基于前一状态的预测
        if self.prev_predicted is not None:
            prediction = self.compute_prediction(self.prev_active_cells)
        else:
            prediction = torch.zeros(batch_size, seq_len, 
                                     self.n_columns, self.n_cells_per_col)
        
        # 2. 确定当前激活细胞
        active_cells = torch.zeros_like(prediction)
        
        # 对每个激活列
        for b in range(batch_size):
            for t in range(seq_len):
                active_cols = active_columns_curr[b, t].nonzero().squeeze(1)
                for col in active_cols:
                    # 检查该列是否有预测的细胞
                    col_predictions = prediction[b, t, col]
                    if col_predictions.sum() > 0:
                        # 正确预测：激活预测的细胞
                        active_cells[b, t, col] = col_predictions
                    else:
                        # 未预测：burst（激活所有细胞）并选择学习细胞
                        active_cells[b, t, col, :] = 1.0  # burst
                        if learn:
                            # 选择最少使用的细胞来学习这个新模式
                            learning_cell = self.burst_column(col.item())
                            self.cell_usage[col, learning_cell] += 1
        
        # 3. 更新状态历史
        self.prev_predicted = prediction
        self.prev_active_cells = active_cells
        
        # 4. 计算预测准确率
        prediction_accuracy = self._compute_accuracy(prediction, active_cells)
        
        return {
            'active_cells': active_cells,
            'prediction': prediction,
            'prediction_accuracy': prediction_accuracy,
            'burst_columns': self._identify_bursts(active_cells)
        }
```

#### 2.3 动态稀疏注意力 (DynamicSparseAttention)

**功能描述**：基于脉冲神经网络的动态注意力机制，替代Transformer的自注意力

```python
class DynamicSparseAttention(torch.nn.Module):
    """
    基于SNN脉冲时序的动态稀疏注意力
    无Transformer，无softmax密集计算
    """
    
    def __init__(self, config):
        super().__init__()
        self.config = config
        self.n_columns = config.n_columns
        self.attention_span = config.attention_span  # 注意力窗口
        self.max_conns = config.max_conns_per_cell  # 每个细胞最大连接数
        
        # 脉冲神经元参数
        self.tau = config.snn_tau  # 膜时间常数
        self.threshold = config.snn_threshold  # 脉冲阈值
        self.refractory = config.snn_refractory  # 不应期
        
        # 可学习的连接权重（稀疏初始化）
        self.connections = self._init_sparse_connections()
        self.stdp_learner = STDPLearner(config)
        
        # 状态追踪
        self.membrane_potential = None
        self.refractory_count = None
    
    def _init_sparse_connections(self):
        """初始化稀疏连接矩阵 ~5%密度"""
        import torch_sparse
        
        n = self.n_columns
        # 随机生成稀疏连接
        nnz = int(n * n * 0.05)  # 5%密度
        row = torch.randint(0, n, (nnz,))
        col = torch.randint(0, n, (nnz,))
        values = torch.randn(nnz) * 0.1
        
        return torch_sparse.SparseTensor(row=row, col=col, value=values, 
                                        sparse_sizes=(n, n))
    
    def forward(self, input_sdr, prev_spikes=None):
        """
        脉冲神经网络前向传播
        input_sdr: [batch, seq_len, n_columns] 稀疏分布式表示
        """
        batch_size, seq_len, _ = input_sdr.shape
        
        # 初始化膜电位和不应期
        if self.membrane_potential is None:
            self.membrane_potential = torch.zeros(batch_size, seq_len, 
                                                  self.n_columns)
            self.refractory_count = torch.zeros_like(self.membrane_potential)
        
        # 脉冲输出序列
        spike_sequence = []
        
        for t in range(seq_len):
            # 1. 不应期衰减
            self.refractory_count[:, t] = torch.clamp(
                self.refractory_count[:, t] - 1, min=0
            )
            
            # 2. 膜电位衰减 (LIF神经元)
            if t > 0:
                decay = torch.exp(-torch.ones_like(self.membrane_potential[:, t-1]) / self.tau)
                self.membrane_potential[:, t] = self.membrane_potential[:, t-1] * decay
            
            # 3. 输入电流注入
            in_current = input_sdr[:, t]
            
            # 4. 稀疏连接的电流传播（优化的稀疏矩阵乘法）
            prev_t = max(0, t-1)
            if prev_spikes is not None and len(prev_spikes) > 0:
                recv_current = self.connections.matmul(prev_spikes[-1].float())
            else:
                recv_current = self.connections.matmul(input_sdr[:, prev_t].float())
            
            # 5. 总电流
            total_current = in_current + 0.3 * recv_current
            
            # 6. 更新膜电位（不应期的神经元不接收电流）
            non_refractory = (self.refractory_count[:, t] == 0).float()
            self.membrane_potential[:, t] += total_current * non_refractory
            
            # 7. 检测脉冲发放
            spikes = (self.membrane_potential[:, t] > self.threshold).float()
            
            # 8. 重置发放脉冲的神经元，设置不应期
            self.membrane_potential[:, t] *= (1 - spikes)  # 重置电位
            self.refractory_count[:, t] = spikes * self.refractory
            
            spike_sequence.append(spikes)
            
            # 9. STDP学习 (如果处于训练模式)
            if self.training and t > 0:
                delta_w = self.stdp_learner.compute_weight_update(
                    spike_sequence[t-1].unsqueeze(1),
                    spikes.unsqueeze(1),
                    self.connections
                )
                self._apply_stdp_update(delta_w)
        
        return {
            'spike_sequence': torch.stack(spike_sequence, dim=1),  # [B, T, C]
            'membrane_potential': self.membrane_potential,
            'attention_map': self._build_attention_map(spike_sequence)
        }
```

### 3. 神经调节器 (NeuralRegulator)

**功能描述**：模拟大脑蓝斑核/去甲肾上腺素系统，调节全局注意力和学习率

```python
class NeuralRegulator(torch.nn.Module):
    """神经调节系统 - 全局状态调节"""
    
    def __init__(self, config):
        super().__init__()
        self.config = config
        
        # 状态跟踪
        self.error_history = []  # 预测误差历史
        self.reward_history = []  # 奖励信号历史
        
        # 基础参数
        self.base_attention_strength = 1.0
        self.base_learning_rate = config.learning_rate
        
        # 自适应参数 (可训练或规则驱动)
        self.adaptation_speed = 0.1
        
        # 内部状态
        self.arousal_level = 0.5  # 唤醒水平 [0,1]
        self.focus_level = 0.5    # 专注水平 [0,1]
        
        # 状态评估网络
        self.state_evaluator = torch.nn.Sequential(
            torch.nn.Linear(5, 32),  # [error, reward, complexity, arousal, focus]
            torch.nn.ReLU(),
            torch.nn.Linear(32, 3)   # [attention_strength, learning_rate, system_bias]
        )
    
    def forward(self, htm_output, prediction_error, task_context=None):
        """
        根据系统状态生成调节信号
        """
        batch_size = htm_output['active_cells'].shape[0]
        
        # 1. 计算系统状态特征
        error_rate = prediction_error.mean().item()
        sequence_complexity = self._compute_complexity(htm_output)
        
        # 更新历史
        self.error_history.append(error_rate)
        if len(self.error_history) > 100:
            self.error_history.pop(0)
        
        # 2. 计算唤醒水平 (基于近期误差趋势)
        error_trend = self._compute_error_trend()
        self.arousal_level = torch.sigmoid(torch.tensor(error_trend * 5 + 0.5))
        
        # 3. 计算专注水平 (基于激活稀疏度)
        activation_sparsity = self._compute_activation_sparsity(htm_output)
        self.focus_level = activation_sparsity
        
        # 4. 构建状态向量
        recent_reward = self.reward_history[-1] if self.reward_history else 0
        state_vector = torch.tensor([
            error_rate,
            recent_reward,
            sequence_complexity,
            self.arousal_level,
            self.focus_level
        ]).unsqueeze(0).repeat(batch_size, 1).float()
        
        # 5. 生成调节信号
        raw_regulation = self.state_evaluator(state_vector)
        
        # 6. 应用约束和归一化
        regulation = self._apply_constraints(raw_regulation)
        
        return RegulationSignal(
            attention_strength=regulation[:, 0],
            learning_rate=regulation[:, 1],
            system1_bias=regulation[:, 2],
            system2_bias=1 - regulation[:, 2],
            arousal_level=self.arousal_level,
            focus_level=self.focus_level,
            prediction_error=error_rate
        )
    
    def _compute_error_trend(self):
        """计算误差趋势：正=误差上升，负=误差下降"""
        if len(self.error_history) < 10:
            return 0
        recent = np.mean(self.error_history[-5:])
        earlier = np.mean(self.error_history[-10:-5])
        return recent - earlier
    
    def _apply_constraints(self, raw_signal):
        """应用物理约束确保信号在合理范围"""
        # attention_strength: [0.3, 2.0]
        attention = 0.3 + 1.7 * torch.sigmoid(raw_signal[:, 0])
        
        # learning_rate: [1e-5, 1e-3]
        lr = 1e-5 + (1e-3 - 1e-5) * torch.sigmoid(raw_signal[:, 1])
        
        # system_bias: [0.1, 0.9] (偏向System1)
        system_bias = 0.1 + 0.8 * torch.sigmoid(raw_signal[:, 2])
        
        return torch.stack([attention, lr, system_bias], dim=1)

@dataclass
class RegulationSignal:
    """调节信号数据结构"""
    attention_strength: torch.Tensor
    learning_rate: torch.Tensor
    system1_bias: torch.Tensor
    system2_bias: torch.Tensor
    arousal_level: float
    focus_level: float
    prediction_error: float
    
    def get_attention_mask(self, base_mask):
        """根据注意力强度调整掩码"""
        return base_mask * self.attention_strength.unsqueeze(-1).unsqueeze(-1)
```

```python
class System1Intuition(torch.nn.Module):
    """
    系统1: 直觉联想系统
    基于快速最近邻检索和统计模式匹配
    无Transformer，使用高效向量检索
    """
    
    def __init__(self, config):
        super().__init__()
        self.config = config
        self.index_dim = config.index_dim  # e.g., 256
        
        # 快速检索引擎 (FAISS)
        self.index = self._init_faiss_index()
        
        # 模式嵌入网络
        self.encoder = torch.nn.Sequential(
            torch.nn.Linear(config.n_columns, 512),
            torch.nn.ReLU(),
            torch.nn.Linear(512, self.index_dim),
            torch.nn.L2Norm(dim=-1)  # 归一化便于相似度计算
        )
        
        # 置信度评估
        self.confidence_estimator = torch.nn.Sequential(
            torch.nn.Linear(self.index_dim + 1, 64),  # 嵌入 + 距离
            torch.nn.ReLU(),
            torch.nn.Linear(64, 1),
            torch.nn.Sigmoid()
        )
        
        # 模式缓存
        self.pattern_cache = {}
        self.max_cache_size = 10000
    
    def forward(self, htm_sparse_output, regulation_signal=None):
        """
        快速直觉推理
        htm_sparse_output: [batch, seq_len, n_columns]
        """
        batch_size, seq_len, _ = htm_sparse_output.shape
        
        # 1. 降维和编码
        flattened = htm_sparse_output.reshape(-1, htm_sparse_output.shape[-1])
        embeddings = self.encoder(flattened)  # [B*T, D]
        embeddings = embeddings.reshape(batch_size, seq_len, self.index_dim)
        
        # 2. 快速最近邻检索
        results = []
        for b in range(batch_size):
            batch_results = []
            for t in range(seq_len):
                vec = embeddings[b, t].cpu().numpy()
                
                # 检查缓存
                cache_key = tuple(vec.round(4))
                if cache_key in self.pattern_cache:
                    batch_results.append(self.pattern_cache[cache_key])
                    continue
                
                # FAISS检索
                distances, indices = self.index.search(
                    vec.reshape(1, -1), 
                    k=self.config.top_k_patterns
                )
                
                # 3. 置信度计算
                avg_distance = np.mean(distances)
                confidence_input = torch.cat([
                    torch.tensor(vec),
                    torch.tensor([avg_distance])
                ]).float()
                confidence = self.confidence_estimator(confidence_input).item()
                
                result = {
                    'matched_indices': indices[0],
                    'distances': distances[0],
                    'confidence': confidence,
                    'avg_similarity': 1 / (1 + avg_distance)
                }
                
                # 更新缓存
                if len(self.pattern_cache) < self.max_cache_size:
                    self.pattern_cache[cache_key] = result
                
                batch_results.append(result)
            results.append(batch_results)
        
        # 4. 生成响应（基于检索到的模式）
        response = self._generate_response(results, embeddings)
        
        return {
            'embeddings': embeddings,
            'retrieval_results': results,
            'response': response,
            'avg_confidence': np.mean([r['confidence'] for br in results for r in br])
        }
    
    def _init_faiss_index(self):
        """初始化FAISS索引用于快速检索"""
        import faiss
        
        index = faiss.IndexHNSWFlat(self.index_dim, 32)  # HNSW用于高速检索
        index.hnsw.efSearch = 64
        index.hnsw.efConstruction = 80
        
        # 预填充常见模式（可在训练中学习）
        n_initial_patterns = 10000
        initial_patterns = np.random.randn(n_initial_patterns, self.index_dim)
        faiss.normalize_L2(initial_patterns)
        index.add(initial_patterns)
        
        return index
```

**功能描述**：慢速、深思熟虑的符号推理，对应前额叶皮层

```python
class System2Reasoning(torch.nn.Module):
    """
    系统2: 符号推理系统
    基于显式规则和多步逻辑推导
    无Transformer，使用符号逻辑引擎
    """
    
    def __init__(self, config):
        super().__init__()
        self.config = config
        self.max_depth = config.max_reasoning_depth  # e.g., 3
        
        # 规则库 (可学习的逻辑规则)
        self.rule_base = RuleBase(config)
        
        # 工作记忆缓冲区
        self.reasoning_buffer = []
        
        # 逻辑嵌入网络
        self.logic_encoder = torch.nn.Sequential(
            torch.nn.Linear(config.n_columns, 256),
            torch.nn.ReLU(),
            torch.nn.Linear(256, 128)
        )
        
        # 推理步骤控制器
        self.step_controller = torch.nn.LSTM(
            input_size=128,
            hidden_size=64,
            num_layers=2,
            batch_first=True
        )
        
        # 步骤决策网络
        self.step_decision = torch.nn.Sequential(
            torch.nn.Linear(64, 3),  # [继续, 得出结论, 回溯]
            torch.nn.Softmax(dim=-1)
        )
        
        # 结论生成器
        self.conclusion_generator = torch.nn.Sequential(
            torch.nn.Linear(128, 256),
            torch.nn.ReLU(),
            torch.nn.Linear(256, config.output_dim)
        )
    
    def forward(self, htm_sparse_output, regulation_signal=None, task_context=None):
        """
        多步符号推理
        """
        batch_size, seq_len, _ = htm_sparse_output.shape
        
        # 1. 编码为逻辑表示
        logic_embeds = self.logic_encoder(htm_sparse_output)  # [B, T, 128]
        
        all_conclusions = []
        all_reasoning_traces = []
        
        for b in range(batch_size):
            # 2. 初始化推理状态
            current_state = logic_embeds[b, -1]  # 取最后一个时间步
            reasoning_trace = []
            facts_extracted = self._extract_facts(htm_sparse_output[b])
            depth = 0
            
            # 3. 多步推理循环
            while depth < self.max_depth:
                reasoning_trace.append({
                    'depth': depth,
                    'state': current_state.detach().cpu().numpy(),
                    'facts': facts_extracted
                })
                
                # 4. 规则匹配与应用
                matched_rules = self.rule_base.match_rules(facts_extracted, current_state)
                
                if not matched_rules:
                    # 无匹配规则，终止推理
                    break
                
                # 5. 应用最相关规则
                best_rule = matched_rules[0]
                new_facts, deduction = self.rule_base.apply_rule(best_rule, facts_extracted)
                
                # 6. 更新状态
                current_state = self._update_state(current_state, deduction)
                facts_extracted = new_facts
                
                # 7. 决定下一步动作
                step_input = current_state.unsqueeze(0).unsqueeze(0)
                _, (h, _) = self.step_controller(step_input)
                step_probs = self.step_decision(h[-1])
                step_action = step_probs.argmax().item()
                
                if step_action == 1:  # 得出结论
                    break
                elif step_action == 2:  # 回溯
                    if len(reasoning_trace) > 1:
                        reasoning_trace.pop()
                        depth -= 1
                        continue
                
                depth += 1
            
            # 8. 生成最终结论
            conclusion = self.conclusion_generator(current_state.unsqueeze(0))
            
            all_conclusions.append(conclusion)
            all_reasoning_traces.append(reasoning_trace)
        
        return {
            'conclusions': torch.cat(all_conclusions, dim=0),
            'reasoning_traces': all_reasoning_traces,
            'rule_activations': self.rule_base.get_activation_counts()
        }
    
    def _extract_facts(self, sequence_data):
        """从稀疏序列中提取事实"""
        # 简化：基于激活模式提取关键事实
        # 实际实现应更复杂，包括实体识别、关系抽取等
        facts = []
        active_positions = (sequence_data.abs() > 0.1).any(dim=-1)
        
        for t, is_active in enumerate(active_positions):
            if is_active:
                activation_pattern = sequence_data[t]
                fact = self._pattern_to_fact(activation_pattern)
                facts.append(fact)
        
        return facts
```

```python
class WorkingMemory:
    """
    工作记忆系统 - 容量有限(4±2个槽位)，激活衰减
    模拟大脑前额叶-顶叶网络
    """
    
    def __init__(self, config):
        self.config = config
        self.capacity = config.working_memory_capacity  # 4±2
        self.decay_rate = config.memory_decay_rate      # 激活衰减率
        self.interference_factor = config.interference_factor
        
        # 记忆槽位
        self.slots = [None] * self.capacity
        self.activation_levels = torch.zeros(self.capacity)
        self.slot_timestamps = torch.zeros(self.capacity)
        
        # 当前时间步
        self.current_step = 0
    
    def store(self, item, item_embedding, priority=1.0):
        """存储项目到工作记忆"""
        self.current_step += 1
        
        # 1. 衰减现有激活
        self.activation_levels *= (1 - self.decay_rate)
        
        # 2. 查找空槽或最低激活槽
        slot_to_use = self._select_slot(priority)
        
        # 3. 存储新项目
        self.slots[slot_to_use] = {
            'item': item,
            'embedding': item_embedding,
            'timestamp': self.current_step
        }
        self.activation_levels[slot_to_use] = priority
        self.slot_timestamps[slot_to_use] = self.current_step
        
        return slot_to_use
    
    def retrieve(self, query_embedding=None, top_k=1):
        """基于激活水平或相似度检索"""
        if query_embedding is None:
            # 返回最激活的项目
            scores = self.activation_levels
        else:
            # 基于嵌入相似度检索
            similarities = torch.zeros(self.capacity)
            for i, slot in enumerate(self.slots):
                if slot is not None:
                    sim = torch.cosine_similarity(
                        query_embedding.flatten(),
                        slot['embedding'].flatten(),
                        dim=0
                    )
                    similarities[i] = sim * (0.5 + 0.5 * self.activation_levels[i])
            scores = similarities
        
        # 获取top-k结果
        top_indices = scores.argsort(descending=True)[:top_k]
        
        results = []
        for idx in top_indices:
            if self.slots[idx] is not None:
                results.append({
                    'slot_idx': idx,
                    'content': self.slots[idx],
                    'activation': self.activation_levels[idx].item(),
                    'score': scores[idx].item()
                })
        
        return results
    
    def refresh(self, slot_idx):
        """刷新指定槽位的激活水平"""
        if 0 <= slot_idx < self.capacity and self.slots[slot_idx] is not None:
            self.activation_levels[slot_idx] = min(
                1.0,
                self.activation_levels[slot_idx] + 0.3
            )
            self.slot_timestamps[slot_idx] = self.current_step
    
    def _select_slot(self, priority):
        """选择要使用的槽位"""
        # 优先选择空槽
        empty_slots = [i for i, s in enumerate(self.slots) if s is None]
        if empty_slots:
            return empty_slots[0]
        
        # 否则替换最低激活的槽
        # 考虑时间因素：旧的槽更容易被替换
        age_penalty = (self.current_step - self.slot_timestamps) / 100.0
        combined_score = self.activation_levels - age_penalty
        
        return combined_score.argmin().item()
```

```python
class EpisodicMemory:
    """
    情景记忆系统 - 时序事件存储与检索
    模拟海马体-内嗅皮层系统
    """
    
    def __init__(self, config):
        self.config = config
        self.max_episodes = config.max_episodes  # e.g., 10000

        self.episode_timestamps = {}
        
        # 时序索引：便于按时间检索
        self.temporal_index = []
        
        # 相似度索引 (FAISS)
        self.index_dim = config.episode_index_dim
        self.index = self._init_index()
    
    def store_episode(self, sequence_data, metadata=None):
        """存储一个情景片段"""
        episode_id = str(uuid.uuid4())
        
        # 计算情节摘要向量用于检索
        episode_summary = self._compute_episode_summary(sequence_data)
        
        # 存储
        self.episodes[episode_id] = {
            'sequence': sequence_data,
            'summary': episode_summary,
            'metadata': metadata or {},
            'timestamp': datetime.now()
        }
        
        # 更新索引
        self._add_to_index(episode_id, episode_summary)
        self.temporal_index.append(episode_id)
        
        # 容量控制
        if len(self.episodes) > self.max_episodes:
            self._forget_oldest()
        
        return episode_id
    
    def retrieve_by_similarity(self, query_summary, top_k=5, time_window=None):
        """按相似度检索情景"""
        # FAISS检索
        distances, indices = self.index.search(
            query_summary.cpu().numpy().reshape(1, -1),
            k=top_k * 2  # 检索更多以应用时间过滤
        )
        
        results = []
        for idx, dist in zip(indices[0], distances[0]):
            episode_id = self._get_episode_id_from_index(idx)
            episode = self.episodes.get(episode_id)
            
            if episode:
                # 时间窗口过滤
                if time_window is not None:
                    age = (datetime.now() - episode['timestamp']).total_seconds()
                    if age > time_window:
                        continue
                
                results.append({
                    'episode_id': episode_id,
                    'similarity': 1 / (1 + dist),
                    'distance': dist,
                    'content': episode
                })
        
        return sorted(results, key=lambda x: x['similarity'], reverse=True)[:top_k]
    
    def pattern_completion(self, partial_sequence):
        """
        模式补全：根据部分线索完成完整情景
        模拟海马体的模式补全功能
        """
        partial_summary = self._compute_episode_summary(partial_sequence)
        
        # 检索最相似的情景
        candidates = self.retrieve_by_similarity(partial_summary, top_k=3)
        
        if not candidates:
            return None
        
        # 从最相似的情景中进行补全
        best_match = candidates[0]['content']
        completed = self._align_and_complete(partial_sequence, best_match['sequence'])
        
        return {
```

```python
class BioGenerator(torch.nn.Module):
    """
    生物启发式生成器 - 无Transformer解码器
    基于稀疏表示的RNN解码
    """
    
    def __init__(self, config):
        super().__init__()
        self.config = config
        self.vocab_size = config.vocab_size
        
        # 解码器：多层RNN + 稀疏投影
        self.decoder_rnn = torch.nn.LSTM(
            input_size=config.decoder_input_dim,
            hidden_size=config.decoder_hidden_dim,
            num_layers=config.decoder_layers,
            bidirectional=False,
            batch_first=True,
            dropout=0.1
        )
        
        # 稀疏投影层 (输入稀疏表示)
        self.sparse_projection = torch.nn.Sequential(
            torch.nn.Linear(config.n_columns + config.decoder_hidden_dim, 1024),
            torch.nn.ReLU(),
            SparsifyActivation(sparsity_target=0.9),
            torch.nn.Linear(1024, 512),
            torch.nn.ReLU()
        )
        
        # 输出层
        self.output_layer = torch.nn.Linear(512, self.vocab_size)
        
        # 前向模型：预测下一个状态
        self.forward_model = ForwardModel(config)
        
        # 规划模块
        self.planner = GenerationPlanner(config)
    
    def forward(self, reasoning_output, memory_context, max_length=50):
        """
        自回归生成文本
        无Transformer，纯RNN+稀疏机制
        """
        batch_size = reasoning_output['conclusions'].shape[0]
        device = reasoning_output['conclusions'].device
        
        # 1. 生成规划
        generation_plan = self.planner.plan(reasoning_output, memory_context)
        
        # 2. 解码状态初始化
        decoder_state = None
        generated_tokens = []
        generation_scores = []
        
        # 初始输入：使用推理结论作为初始状态
        current_input = self._prepare_initial_input(reasoning_output, memory_context)
        
        for step in range(max_length):
            # 3. RNN解码
            decoder_out, decoder_state = self.decoder_rnn(
                current_input.unsqueeze(1),
                decoder_state
            )
            decoder_out = decoder_out.squeeze(1)
            
            # 4. 获取HTM/推理上下文 (简化：这里使用规划信息)
            htm_context = self._get_htm_context(reasoning_output, step)
            
            # 5. 稀疏投影与生成
            combined = torch.cat([decoder_out, htm_context], dim=-1)
            sparse_features = self.sparse_projection(combined)
            
            # 6. 前向模型预测与验证
            future_prediction = self.forward_model.predict(sparse_features, step)
            adjusted_features = self.forward_model.validate_and_adjust(
                sparse_features,
                future_prediction
            )
            
            # 7. 输出token预测
            logits = self.output_layer(adjusted_features)
            next_token = torch.argmax(logits, dim=-1)
            
            # 8. 计算置信度分数
            score = torch.softmax(logits, dim=-1).max(dim=-1)[0]
            
            generated_tokens.append(next_token)
            generation_scores.append(score)
            
            # 9. 准备下一步输入
            current_input = self._token_to_decoder_input(next_token)
            
            # 10. 终止检查 (EOS token)
            if (next_token == self.config.eos_token_id).all():
                break
        
        # 整理输出
        output_sequence = torch.stack(generated_tokens, dim=1)
        output_scores = torch.stack(generation_scores, dim=1)
        
        return {
            'token_ids': output_sequence,
            'scores': output_scores,
            'plan': generation_plan,
            'forward_predictions': self.forward_model.get_predictions_history()
        }

class ForwardModel(torch.nn.Module):
    """
    前向模型：预测动作的后果，用于生成质量控制
    """
    
    def __init__(self, config):
        super().__init__()
        self.config = config
        
        # 预测网络
        self.predictor = torch.nn.Sequential(
            torch.nn.Linear(config.n_columns + 1, 512),  # 状态 + 步长
            torch.nn.ReLU(),
            torch.nn.Linear(512, 256),
            torch.nn.ReLU(),
            torch.nn.Linear(256, config.n_columns)
        )
        
        # 质量评估
        self.quality_scorer = torch.nn.Sequential(
            torch.nn.Linear(config.n_columns * 2, 64),
            torch.nn.ReLU(),
            torch.nn.Linear(64, 1),
            torch.nn.Sigmoid()
        )
        
        self.prediction_history = []
    
    def predict(self, current_state, step):
        """预测下一个状态"""
        step_tensor = torch.tensor([step], device=current_state.device).repeat(
            current_state.shape[0], 1
        )
        predictor_input = torch.cat([current_state, step_tensor.float()], dim=-1)
        prediction = self.predictor(predictor_input)
        
        self.prediction_history.append(prediction)
        return prediction
    
    def validate_and_adjust(self, current_state, prediction):
        """验证当前状态与预测的一致性，必要时调整"""
        quality_input = torch.cat([current_state, prediction], dim=-1)
        quality_score = self.quality_scorer(quality_input)
        
        # 如果质量低于阈值，使用预测辅助
        adjustment_mask = (quality_score < self.config.quality_threshold).float()
        adjustment_mask = adjustment_mask.unsqueeze(-1)
        
        # 调整后的状态：当前状态 + 预测残差
        adjusted = current_state * (1 - adjustment_mask) + \
                   (current_state + 0.3 * prediction) * adjustment_mask
        
        return adjusted
```

### 3. 动态稀疏注意力 (SNN) - 脉冲神经网络

**功能描述**：基于脉冲时序的动态注意力机制

**核心特性**：
- 脉冲时序依赖可塑性(STDP)学习
- 动态关注预测误差较大的区域
- 稀疏激活模式（激活率 < 10%）
- 能量高效的事件驱动计算

```python
class SpikingAttention(torch.nn.Module):
    """脉冲神经网络注意力，无Transformer"""
    
    def __init__(self, config):
        super().__init__()
        self.config = config
        self.n_neurons = config.n_columns  # 与HTM列数对应
        self.tau = config.tau  # 膜时间常数 (20ms)
        self.v_threshold = config.v_threshold  # 脉冲阈值
        self.refractory_period = config.refractory  # 不应期 (5ms)
        
        # STDP参数
        self.tau_plus = 20.0
        self.tau_minus = 20.0
        self.A_plus = 0.01
        self.A_minus = 0.01
        
        # 可训练连接权重
        self.forward_weights = torch.nn.Parameter(
            torch.randn(self.n_neurons, self.n_neurons) * 0.01
        )
        self.recurrent_weights = torch.nn.Parameter(
            torch.randn(self.n_neurons, self.n_neurons) * 0.01
        )
        
        # 稀疏掩码：只有2%连接
        self.sparse_mask = (torch.rand(self.n_neurons, self.n_neurons) < 0.02).float()
        self.forward_weights.data *= self.sparse_mask
        self.recurrent_weights.data *= self.sparse_mask
        
    def forward(self, htm_activations, prediction_error):
        """
        htm_activations: HTM层输出 [batch, seq_len, n_columns]
        prediction_error: 预测误差信号 [batch, seq_len, n_columns]
        """
        batch_size, seq_len, _ = htm_activations.shape
        
        # 1. 转换为脉冲输入
        input_spikes = self._rate_to_spikes(htm_activations, seq_len)
        error_spikes = self._rate_to_spikes(prediction_error, seq_len)
        
        # 2. 初始化膜电位和脉冲记录
        v_membrane = torch.zeros(batch_size, self.n_neurons)
        spike_times = torch.zeros(batch_size, seq_len, self.n_neurons)
        refractory_counter = torch.zeros(batch_size, self.n_neurons)
        
        # 3. 脉冲模拟
        for t in range(seq_len):
            # 不应期更新
            refractory_counter = torch.max(torch.zeros_like(refractory_counter), 
                                          refractory_counter - 1)
            not_refractory = (refractory_counter == 0).float()
            
            # 膜电位更新（泄漏积分）
            dv_dt = (-v_membrane / self.tau)  # 泄漏项
            
            # 输入电流
            input_current = torch.einsum('bi,ij->bj', input_spikes[:, t], 
                                        self.forward_weights)
            
            # 递归电流
            if t > 0:
                recurrent_current = torch.einsum('bi,ij->bj', spike_times[:, t-1], 
                                                self.recurrent_weights)
            else:
                recurrent_current = 0
            
            # 误差增强电流
            error_current = torch.einsum('bi,ij->bj', error_spikes[:, t], 
                                        self.forward_weights) * 2.0
            
            # 总膜电位更新
            v_membrane += dv_dt + (input_current + recurrent_current + error_current) * not_refractory
            
            # 检测脉冲发放
            spikes = (v_membrane >= self.v_threshold).float()
            
            # 重置发放脉冲的神经元
            v_membrane = v_membrane * (1 - spikes)  # 发放后重置
            refractory_counter = refractory_counter + spikes * self.refractory_period
            
            spike_times[:, t] = spikes
        
        # 4. 计算注意力权重
        spike_rates = spike_times.mean(dim=1)  # [batch, n_neurons]
        
        # 5. STDP学习更新权重
        if self.training:
            self._apply_stdp(spike_times)
        
        # 6. 应用注意力到HTM激活
        attention_map = torch.softmax(spike_rates * 5.0, dim=-1)  # 温度参数
        attended = htm_activations * attention_map.unsqueeze(1)
        
        return {
            'spike_times': spike_times,
            'spike_rates': spike_rates,
            'attention_map': attention_map,
            'attended_activations': attended
        }
    
    def _rate_to_spikes(self, rates, seq_len):
        """将激活率转换为泊松脉冲序列"""
        batch_size = rates.shape[0]
        
        # 确保rates是非负的
        rates_pos = torch.relu(rates)
        rates_scaled = rates_pos / rates_pos.max(dim=-1, keepdim=True).values.clamp(min=1e-8)
        
        # 生成泊松分布的脉冲
        spikes = torch.bernoulli(rates_scaled.unsqueeze(1).repeat(1, seq_len, 1) * 0.3)
        return spikes
    
    def _apply_stdp(self, spike_times):
        """应用STDP规则更新权重"""
        batch_size, seq_len, n_neurons = spike_times.shape
        
        # 简化的STDP实现
        for t_post in range(1, seq_len):
            for t_pre in range(max(0, t_post - 40), t_post):  # STDP时间窗40步
                delta_t = t_post - t_pre
                
                # 找出此时刻的脉冲发放神经元
                pre_spikes = spike_times[:, t_pre]  # [batch, n]
                post_spikes = spike_times[:, t_post]  # [batch, n]
                
                # 计算突触前到突触后的相关性
                # [batch, n_pre, 1] * [batch, 1, n_post] -> [batch, n_pre, n_post]
                correlation = torch.einsum('bi,bj->bij', pre_spikes, post_spikes)
                correlation = correlation.mean(dim=0)  # batch平均
                
                # 根据时间差应用LTP/LTD
                if delta_t > 0:  # 前突触先发放 -> LTP
                    ltp = self.A_plus * torch.exp(-delta_t / self.tau_plus)
                    self.forward_weights.data += correlation * ltp * self.sparse_mask
                else:  # 后突触先发放 -> LTD
                    delta_t_abs = -delta_t
                    ltd = self.A_minus * torch.exp(-delta_t_abs / self.tau_minus)
                    self.forward_weights.data -= correlation * ltd * self.sparse_mask
        
        # 截断权重范围
        self.forward_weights.data.clamp_(-1.0, 1.0)
        # 保持稀疏性
        self.forward_weights.data *= self.sparse_mask
```

```python
        深度符号推理：问题分析 -> 推理规划 -> 多步演绎 -> 验证

            start_state=problem_rep['initial_state'],

                continue
                
            intermediate_results.append({
                'step': step,
                'action': action,
                'state': current_state,
                'next_state': next_state
            })

            if self._goal_reached(current_state, problem_rep['goal_state']):
                break

        conclusion = self._generate_conclusion(current_state)
        explanation = self._generate_explanation(intermediate_results)
        
        # 6. 置信度评估（基于推理步骤的置信度）

            intermediate_results, current_state

            'conclusion': conclusion,
            'intermediate_results': intermediate_results,
            'reasoning_path': reasoning_plan,
            'explanation': explanation,
```

```python
class WorkingMemory:
    """
    生物启发式工作记忆
    基于激活衰减和容量限制
    """
    def __init__(self, config):
        self.config = config
        self.capacity = config.working_memory_capacity  # 默认为4
        self.decay_rate = config.memory_decay_rate  # 激活衰减率
        
        # 记忆槽位：存储内容、激活强度、时间戳
        self.slots = []  # list of MemorySlot
        self.last_refresh = time.time()
        
    @dataclass
    class MemorySlot:
        content: Any  # 可以是张量、符号、对象引用
        activation: float  # 激活强度
        decay_rate: float  # 个性化衰减率
        timestamp: float  # 最后访问时间
        importance: float  # 重要性评分
        
    def add(self, content, importance=0.5, decay_rate=None):
        """添加内容到工作记忆，必要时替换最不活跃的槽位"""
        if decay_rate is None:
            decay_rate = self.decay_rate
            
        # 先衰减已有内容的激活
        self._decay_activations()
        
        # 检查是否有空闲槽位
        if len(self.slots) < self.capacity:
            # 添加新槽位
            self.slots.append(self.MemorySlot(
                content=content,
                activation=1.0,
                decay_rate=decay_rate,
                timestamp=time.time(),
                importance=importance
            ))
        else:
            # 替换激活最低的槽位（考虑重要性）
            slot_scores = [s.activation * s.importance for s in self.slots]
            replace_idx = np.argmin(slot_scores)
            
            # 只在新内容更重要时替换
            if importance > self.slots[replace_idx].importance:
                self.slots[replace_idx] = self.MemorySlot(
                    content=content,
                    activation=1.0,
                    decay_rate=decay_rate,
                    timestamp=time.time(),
                    importance=importance
                )
                
    def retrieve(self, query=None, top_k=2):
        """检索工作记忆内容，基于激活度和相似度"""
        self._decay_activations()
        
        if not self.slots:
            return []
            
        if query is None:
            # 返回最激活的内容
            sorted_slots = sorted(self.slots, key=lambda x: x.activation, reverse=True)
            results = [(s.content, s.activation) for s in sorted_slots[:top_k]]
        else:
            # 基于查询的相似性检索
            scores = self._compute_similarity(query, self.slots)
            # 结合激活度
            combined_scores = [s * slot.activation 
                             for s, slot in zip(scores, self.slots)]
            # 排序返回
            sorted_pairs = sorted(zip(combined_scores, self.slots), reverse=True)
            results = [(s.content, score) for score, s in sorted_pairs[:top_k]]
            
        # 刷新被检索内容的激活
        for result in results:
            for slot in self.slots:
                if slot.content is result[0]:
                    slot.activation = min(1.0, slot.activation * 1.5)
                    slot.timestamp = time.time()
                    
        return results
    
    def _decay_activations(self):
        """基于时间衰减激活强度"""
        current_time = time.time()
        for slot in self.slots:
            time_passed = current_time - slot.timestamp
            decay_factor = np.exp(-time_passed * slot.decay_rate)
            slot.activation *= decay_factor
            slot.timestamp = current_time
            
        # 清理激活过低的槽位
        self.slots = [s for s in self.slots if s.activation > 0.01]
```

```python
class BioGenerator(BaseModule):
    """
    生物启发式生成器模块
    
    ✅ 完全替代Transformer解码器的纯生物启发架构：
    ┌─────────────────────────────────────────────────────┐
    │  [输入token/embeds] → [SDR编码] → [HTM时序记忆]     │
    │                                              ↓      │
    │  [token输出] ← [SDR解码] ← [上下文调制] ← [序列预测] │
    └─────────────────────────────────────────────────────┘
    
    核心特性：
    - 无自注意力 / 无Transformer组件
    - 稀疏激活：~2%神经元激活，能耗降低~90%
    - 在线学习：支持持续学习新序列模式
    - 可解释性：SDR重叠直接表示语义相似度
    """
    
    def generate(self, input_ids, max_new_tokens=20):
        """自回归生成：完全基于HTM时序记忆"""
        current_ids = input_ids
        
        for _ in range(max_new_tokens):
            # 1. token→SDR转换
            sdr_result = self.sdr_vocabulary(current_ids, mode='token_to_sdr')
            
            # 2. HTM序列预测
            seq_result = self.sequence_predictor(sdr_result['column_activations'])
            
            # 3. 输出SDR→token概率
            token_result = self.sdr_vocabulary(
                seq_result['sequence_output'], 
                mode='sdr_to_token'
            )
            
            # 4. 采样下一个token
            next_token = torch.argmax(token_result['token_logits'][:, -1:, :], dim=-1)
            current_ids = torch.cat([current_ids, next_token], dim=1)
        
        return current_ids
```

| **可解释性** | 黑箱 | SDR语义透明 |
| **能耗效率** | 基准 | ~10倍提升 |

## 训练范式（纯生物启发式，无Transformer预训练）

### 训练阶段总览

| 阶段 | 名称 | 学习机制 | 数据要求 | 目标 |
|------|------|----------|----------|------|
| 1 | 稀疏表示学习 | 空间池化 + Hebbian学习 | 无标注文本 | 建立有效稀疏编码 |
| 2 | 时序预测学习 | 时序记忆 + 预测误差 | 无标注序列 | 学习序列依赖 |
| 3 | SNN脉冲学习 | STDP + 误差调制 | 无标注/自监督 | 动态注意力形成 |
| 4 | 神经调节校准 | 奖励学习 + 自适应 | 验证集/小样本 | 系统动态平衡 |
| 5 | 双系统推理 | 监督学习 + 规则归纳 | 标注任务数据 | 任务特定推理 |
| 6 | 记忆巩固 | 离线回放 + 记忆迁移 | 混合数据集 | 知识整合迁移 |

**总计**：单一端到端训练范式，多层级递进学习

### 训练算法：分层时序预测学习 (HTPL)

```python
def bio_train(model, data_loader, config, device):
    """
    纯生物启发式训练主循环
    无Transformer，无反向传播（或仅稀疏反向传播）
    
    基于：预测误差学习 + Hebbian更新 + 神经调节
    """
    model.train()
    data_adapter = TransformerDataAdapter(config.tokenizer_name)
    
    # 训练状态跟踪
    training_metrics = {
        'sparsity': [],
        'prediction_accuracy': [],
        'prediction_error': [],
        'reward': [],
        'modes': []
    }
    
    total_batches = len(data_loader)
    
    for epoch in range(config.num_epochs):
        epoch_metrics = defaultdict(list)
        
        for batch_idx, batch in enumerate(data_loader):
            # --------------------------
            # 阶段1: 数据适配（兼容Transformer格式）
            # --------------------------
            # 支持从token ids或原始文本输入
            if 'text' in batch:
                encoded = data_adapter.encode_batch(batch['text'])
            else:
                encoded = batch  # 假设已是token ids格式（兼容）
                
            # 转换为稀疏输入
            htm_input = data_adapter.to_htm_input(encoded)
            
            # --------------------------
            # 阶段2: 前向传播
            # --------------------------
            # 2.1 输入处理
            embeddings = model.embedding_converter(
                input_ids=encoded.get('input_ids'),
                input_embeddings=htm_input.get('dense')
            )
            
            # 2.2 感觉门控
            gated = model.sensory_gating(
                embeddings,
                attention_mask=encoded.get('attention_mask'),
                task_context=batch.get('task_type')
            )
            
            # 2.3 多粒度编码
            encoded_features = model.multigranular_encoder(
                gated['gated_activations']
            )
            
            # 2.4 HTM空间池化
            sp_output = model.spatial_pooler(
                encoded_features['combined'],
                learn=True
            )
            
            # 2.5 HTM时序记忆与预测
            tm_output = model.temporal_memory(
                active_columns_prev=sp_output['active_columns'][:, :-1] if sp_output['active_columns'].shape[1] > 1 else None,
                active_columns_curr=sp_output['active_columns'],
                learn=True
            )
            
            # 2.6 SNN动态注意力
            snn_output = model.spiking_attention(
                sp_output['sparse_output'],
                prediction_error=1.0 - tm_output['prediction_accuracy'].mean(dim=[1, 2])
            )
            
            # 2.7 神经调节
            reg_signal = model.neural_regulator(
                prediction_error=1.0 - tm_output['prediction_accuracy'],
                spike_rates=snn_output['spike_rates'],
                task_complexity=batch.get('complexity', 0.5)
            )
            
            # --------------------------
            # 阶段3: 生物启发式学习更新
            # --------------------------
            # 3.1 计算奖励信号（基于预测准确率）
            reward = tm_output['prediction_accuracy'].mean().item()
            epoch_metrics['reward'].append(reward)
            epoch_metrics['prediction_accuracy'].append(reward)
            
            # 3.2 计算稀疏度指标
            sparsity = 1.0 - (sp_output['active_columns'].mean() / config.htm.activation_rate)
            epoch_metrics['sparsity'].append(sparsity.item())
            
            # 3.3 应用神经调节到学习率（Hebbian更新已在各模块内部完成）
            # 注意：HTM/SNN学习在forward过程中已执行（本地Hebbian学习）
            
            # 3.4 如果有标签，进行稀疏梯度更新（仅监督部分）
            if 'labels' in batch or 'label' in batch:
                # 推理核心
                system1_out = model.system1(
                    snn_output['attended_activations'],
                    reg_signal
                )
                system2_out = model.system2(
                    snn_output['attended_activations'],
                    reg_signal,
                    task_type=batch.get('task_type')
                )
                
                # 双系统融合
                fused = model.dual_fusion(system1_out, system2_out, reg_signal)
                
                # 任务损失（仅稀疏反向传播，只更新激活的连接）
                loss = compute_task_loss(fused, batch, config.task_type)
                epoch_metrics['task_loss'].append(loss.item())
                
                # 稀疏梯度更新
                sparse_backward(model, loss, reg_signal.learning_rate)
            
            # --------------------------
            # 阶段4: 指标跟踪与日志
            # --------------------------
            epoch_metrics['mode'].append(reg_signal.mode)
            epoch_metrics['burst_rate'].append(tm_output['bursting'].mean().item())
            
            # 进度日志
            if batch_idx % config.log_interval == 0:
                avg_acc = np.mean(epoch_metrics['prediction_accuracy'][-10:])
                avg_sparsity = np.mean(epoch_metrics['sparsity'][-10:])
                print(f"Epoch {epoch+1}/{config.num_epochs}, "
                      f"Batch {batch_idx}/{total_batches}, "
                      f"Pred_Acc: {avg_acc:.4f}, "
                      f"Sparsity: {avg_sparsity:.4f}, "
                      f"Mode: {reg_signal.mode}")
        
        # --------------------------
        # 阶段5: 离线记忆巩固（Epoch结束后）
        # --------------------------
        if config.enable_memory_consolidation and epoch > 0:
            print("Performing memory consolidation...")
            consolidation_loss = model.memory_system.consolidate(
                model.temporal_memory.prev_active_cells 
                if hasattr(model.temporal_memory, 'prev_active_cells') else None
            )
            epoch_metrics['consolidation_loss'].append(consolidation_loss)
        
        # --------------------------
        # 阶段6: 检查点与早停
        # --------------------------
        save_training_checkpoint(model, epoch, epoch_metrics, config)
        
        # 早停检查（基于预测准确率）
        current_acc = np.mean(epoch_metrics['prediction_accuracy'])
        if early_stopping_check(current_acc, config):
            print(f"Early stopping triggered at epoch {epoch+1}")
            break
    
    return training_metrics
```

### 1. 纯生物启发架构
